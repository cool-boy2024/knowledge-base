## 🛠 How to Use

### Locally with Obsidian
1. Install [Obsidian](https://obsidian.md) (free)
2. "Open local vault" → select this directory
3. `⌘+G` for Graph View, `⌘+P` for command palette

### Browse on GitHub
Mermaid diagrams, bidirectional links, and tables all render directly.

### Add new notes via Claude
Just tell Claude Code: "**Summarize this video/article/book: [URL]**" — Claude will:
1. Fetch content
2. File it under correct PARA folder
3. Generate Mermaid diagrams + bidirectional links
4. Produce bilingual versions (per [vault-bilingual-rule](file://~/.claude/memory/vault-bilingual-rule.md))
5. Commit + push to this repo
6. Run `graphify` to refresh the brain graph
